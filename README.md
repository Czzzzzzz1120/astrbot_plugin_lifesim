# astrbot_plugin_lifesim

模拟人生文字游戏插件，基于 [AstrBot](https://github.com/Sora23333/AstrBot) 框架。使用大模型驱动叙事，玩家创建角色后，AI 会根据世界观、物种、年龄和过往经历实时生成连贯剧情，经历从出生到死亡的完整人生模拟。

支持私聊和群聊，所有 11 条指令以 `人生` 为前缀。可在 QQ 群中多人同时进行，每人独立存档。

---

## 功能特性

### 核心系统

- **物种系统** — 创建角色时选择物种（人类或自定义），AI 根据世界观生成参考示例。自定义物种贯穿全部叙事和事件，确保对话与世界观统一
- **大模型叙事** — 支持 AstrBot 内置模型或手动配置 OpenAI 兼容 API，无模型时也可游玩（使用预设文本）
- **真实属性变化** — 代码预决定属性变化，AI 围绕变化写叙事，【体质+1/颜值-1】100% 真实有效，叙事与变化永远一致
- **5 属性无上限** — 体质、智力、颜值、快乐、家境，下限 1、无上限，高属性角色可达「超凡入圣」级评价

### 游戏内容

- **8 个预设世界观 + 任意自定义** — 武侠江湖、都市白领、仙侠修真、末日生存、古代宫廷、奇幻冒险、赛博朋克、海岛求生
- **30 种天赋** — 按稀有度加权随机出现，创建时 5 选 3（含 `早慧`：16岁前成长+50%、`大器晚成`：40岁属性翻倍）
- **8 种出身背景** — 命名后随机分配，含 AI 生成的出身描述
- **15 种重要事件类型** — 每十年至少 2 个，AI 动态生成场景叙事 + 3 个选项 + 1 个自定义输入
- **风险评估** — 每个选项标注 🟢低/🟡中/🔴高 风险，高风险可能遭受属性惩罚

### AI 系统

- **属性→叙事联动** — 代码随机属性变化 → 送 AI 写一句叙事（≤40字），【】由代码拼接，叙事与变化一致
- **上下文连贯** — AI 生成事件时会参考近期经历和过往抉择，剧情不脱节
- **智能死亡判定** — AI 根据属性值、年龄、事件背景综合判断死因，死亡原因有画面感（非预设文案）
- **世界观一致性** — 所有提示词注入世界观描述、物种和身份，叙事不偏离设定

### 人生结算

- **「人生重开模拟器」风格结算面板** — 属性可视化进度条（▮▯）、属性等级评语、幽默人生总评（SSS~F 各有调侃文案）
- **享年分段评价** — 高寿善终 / 花甲之年 / 英年早逝 / 幼年夭折
- **AI 人生总结** — 结合物种、身份、天赋和关键事件，生成回顾文案

### 群聊适配

- 每位玩家独立存档，`@` 回复，自动 `📖 角色名` 前缀区分不同玩家的故事
- `人生排行` 查看群内玩家进度

## 安装

### 方式一：插件市场

AstrBot WebUI → 插件市场 → 搜索 `astrbot_plugin_lifesim` → 一键安装。

### 方式二：手动安装

```bash
cd AstrBot/data/plugins
git clone https://github.com/Czzzzzzz1120/astrbot_plugin_lifesim.git
pip install openai
```

安装后在 AstrBot WebUI 中重载插件。

---

## 配置

AstrBot WebUI → 插件管理 → `astrbot_plugin_lifesim` → 设置：

| 配置项 | 说明 |
|--------|------|
| **对话模型** | 从 AstrBot 已添加的模型中选择（优先级最高，推荐使用） |
| 手动配置 - API Key | 未选择上方模型时生效，输入 OpenAI 兼容 API Key |
| 手动配置 - API 地址 | 默认 `https://api.openai.com/v1`，可改为 DeepSeek 等 |
| 手动配置 - 模型名称 | 如 `gpt-4o`、`deepseek-chat` 等 |

> 不配置模型也可游玩，但叙事将使用预设文本，不会由 AI 生成。

## 指令列表

所有指令均以 `人生` 为前缀，另有 `rl` 前缀英文别名方便快速输入。

| 指令 | 别名 | 说明 |
|------|------|------|
| `人生开启 <世界观>` | `rlstart` `rlks` | 开始新游戏（预设或自定义世界观均可） |
| `人生世界观` | `rlworld` `rlsj` | 查看可选世界观列表 |
| `人生选择 <选项>` | `rlchoose` `rlxz` | 选择物种 / 性别 / 输入名字 / 天赋 / 重要事件选项 / 自定义输入 |
| `人生加点 <属性> <点数>` | `rlpoint` `rljd` | 分配属性点（10点自由分配） |
| `人生加点 3 3 2 2` | — | 快速分配（五项顺序：体质/智力/颜值/快乐/家境） |
| `人生加点 随机` | — | 随机分配全部剩余点数 |
| `人生继续` | `rlnext` `rljx` | 推进人生（自动推进 5 年，遇重要事件停下等待选择） |
| `人生天赋` | `rltalent` `rltf` | 查看当前天赋 |
| `人生属性` | `rlattr` `rlsx` | 查看当前属性面板 |
| `人生状态` | `rlstatus` `rlzt` | 查看完整游戏状态（含物种/世界观/身份/天赋） |
| `人生重置` | `rlreset` `rlcz` | 放弃当前游戏，任何阶段均可使用 |
| `人生排行` | `rlrank` `rlph` | 查看群内玩家游戏进度排行（仅群聊） |
| `人生帮助` | `rlhelp` `rlh` | 显示帮助信息 |

## 游戏流程

```
人生开启 <世界观>
    ↓
━━ 角色创建 ━━
① 选择物种（AI 根据世界观生成示例，可选人类或自定义）
② 选择性别（1.男 2.女 3.其他）
③ 输入角色名字 → AI 生成出身背景（含物种和世界观）
④ 选择天赋（随机 5 选 3，附带属性加成）
⑤ 自由分配 10 点属性
    ↓
━━ 人生模拟 ━━
⑥ 0岁出生（AI 生成含物种和世界观的出生场景）
⑦ 人生继续 → 自动推进 5 年，遇重要事件停下
⑧ 重要事件：AI 生成场景 + 🟢🟡🔴风险选项，可选 1/2/3 或自定义输入
⑨ AI 综合判断死亡（属性过低/年龄过高/高风险选择等）
    ↓
━━ 人生落幕 ━━
⑩ 角色死亡 → 属性可视化结算 + 幽默人生总评 + AI 人生总结
```

## 属性系统

| 属性 | 说明 | 对寿命的影响 |
|------|------|-------------|
| 体质 | 影响体力、健康 | 每+1 → 寿命+2 |
| 智力 | 影响学习、决策 | — |
| 颜值 | 影响社交、人际关系 | — |
| 快乐 | 影响生活满意度 | — |
| 家境 | 影响资源和运气 | 每+1 → 寿命+1 |

- 初始均为 5 点，通过出身背景（随机）、天赋、自由加点、重要事件和年龄增长持续提升
- 无上限，仅保底 1 点
- 预计寿命 = 75 + 体质加成 + 家境加成 + 天赋加成 + 随机波动（-5~+5）
- 属性等级评语（无上限版本）：

| 范围 | 评语 |
|------|------|
| ≥15 | 金刚不坏 / 天才学霸 / 绝世容颜 / 极乐逍遥 / 富甲天下 |
| ≥9 | 身体不错 / 头脑灵光 / 略有姿色 / 知足常乐 / 小康之家 |
| ≥7 | 凑合 / 普通人智商 / 路人脸 / 平平淡淡 / 勉强温饱 |
| ≥5 | 体弱多病 / 脑子不太灵 / 路人甲 / 苦大仇深 / 一贫如洗 |
| <5 | 体弱多病 / 脑子不太灵 / 路人甲 / 苦大仇深 / 一贫如洗 |

## 重要事件

每十年至少 2 个重要事件，从 1 岁分布到预计寿命之后，15 种事件类型覆盖全人生阶段：

天赋觉醒 → 童年危机 → 贵人相助 → 少年磨难 → 命运转折 → 情感风暴 → 事业起步 → 重大抉择 → 危机降临 → 中年觉醒 → 命运再次考验 → 人生巅峰 → 传承之责 → 暮年危机 → 人生终章

每个事件由 AI 根据物种、身份、属性和近期经历动态生成场景叙事和选项：

```
⚡ 命运转折 ·25岁
收到一封密信，远房叔父留下巨额遗产，需穿越匪患横行的黑风岭。
1.[冒死穿越黑风岭]🔴 家境+3（40%概率受惩罚）
2.[雇镖师护卫同行]🟡 智力+2 颜值+1（20%概率额外奖励）
3.[放弃遗产留在小镇]🟢 家境+2
4.自定义
人生选择 <1/2/3> →
```

## 人生评价

角色死亡后展示「人生重开模拟器」风格结算面板：

### 评分规则

| 项目 | 计算 |
|------|------|
| 属性分 | 五项属性总和 |
| 天赋分 | 天赋数 × 5 |
| 抉择分 | 重要抉择次数 × 3 |
| 年龄分 | 存活年数 × 0.5 |

等级：SSS(≥180) → SS(≥145) → S(≥110) → A(≥85) → B(≥60) → C(≥40) → D(≥25) → F(<25)

## 物种系统

创建角色时在性别之前选择物种：

- AI 根据所选世界观自动生成 3~5 个参考物种示例
- 选择 `1` → 人类；输入其他文字 → 自定义物种（如「精灵」「丘丘人」「史莱姆」）
- 自定义物种内容注入全部后续 AI 提示词，确保叙事与物种一致

## 群聊功能

- 每位玩家独立存档，互不干扰
- 所有回复自动 `@` 发送者
- 个人剧情带 `📖 角色名` 前缀区分不同玩家的故事
- `🐾 物种` 标识显示在故事顶部
- `📖 角色名.的故事\n` 前缀贯穿整个游戏流程
- `人生排行` 按年龄降序列出群内所有玩家进度
